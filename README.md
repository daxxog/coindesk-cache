# CoindeskCache

  Cache server for the coindesk api.

  [![Build Status][travis-image]][travis-url]

Install
-------
stable
```bash
npm install -g coindesk-cache
```
edge
```bash
npm install -g https://github.com/daxxog/coindesk-cache/tarball/master
```
